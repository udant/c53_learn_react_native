Wikipedia - React Native

React Native uses a language called JSX. JSX is a mix of both HTML and JAVASCRIPT.

FUNCTIONS AND TAGS:

Render();--The Render function is used to display whatever components are returned by it.

Text--The Text tag is used to write whatever is written in between the two tags.

Button--The Button tag is used to create a button.Its properties are color(used to give the button a color), title(used to write something on the button) and onpress(used to do something after button is clicked).

View--The View tag is used to create a button.Its property is style(used to beautify the project).
